# wykiller
把你的网易启动器变成离线启动器

网易杀手 wykiller v1
简介：自动跳过网易mc的界面覆盖，并自动注入你的mod。
使用方法：使用前请将mod放入modx文件夹中，modx文件夹和wykiller.py必须在同一目录下
        想玩那个版本就开哪个版本的存档，仅支持java版!!!!!

更新日志：
v1.0.0:把所有代码放在一个文件里了，功能比较简单，主要是删除网易mc的界面覆盖，并把modx里的mod复制到mods里，本质上算是一个离线启动器。
v1.0.1:调整代码健壮性,多平台通用性。
v1.0.2:增加设置文件,增加了多语言支持，默认英文，中文用户请修改setting.json里的lang为zh。
